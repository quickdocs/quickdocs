# Quickdocs

- http://quickdocs.org/

## Setup

    $ make
    $ shly update-dist

## Updating

    $ shly ql:update-all-dists --prompt nil
    $ shly update-dist

## Start a server

    $ shly start --port 8080 --server :fcgi
    $ APP_ENV=production start_server --port 8080 -- shly start --server :fcgi
